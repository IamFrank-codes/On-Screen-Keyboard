# On-Screen Laptop Keyboard

This project is a responsive **laptop-style on-screen keyboard** built using **HTML, CSS, and JavaScript**. It simulates a standard **QWERTY laptop keyboard** and allows users to type text by clicking on on-screen keys.

The keyboard includes essential keys found on a real laptop keyboard, such as **Shift**, **Caps Lock**, **Backspace**, **Enter**, and **Space**, providing a realistic typing experience.

This project focuses on **JavaScript DOM manipulation**, **event handling**, and **responsive user interface design**, making it suitable for learning purposes and as a front-end portfolio project.
